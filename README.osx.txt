`rRootage' is a vertical shooter game, created by ABA Games. You can
download that Windows version from the ABA Games.

    http://www.asahi-net.or.jp/~cs8k-cyu/index_e.html

This is a Mac OS X version of rRootage. Please report to me when you
find some bugs.

Some options can be specified on the Terminal.app:

    /Applications/rRootage.app/Contents/MacOS/rr -window

For more information, please read readme_e.txt.
